### Задание 2
Проведите сканирование Metasploitable в режимах SYN, FIN, Xmas, UDP.

Запишите сеансы сканирования в Wireshark.

Ответьте на следующие вопросы:

Чем отличаются эти режимы сканирования с точки зрения сетевого трафика?
Как отвечает сервер?
Приведите ответ в свободной форме.

#### ОТВЕТ:

1. SYN сканирование
Отправляет TCP-пакет с флагом SYN. Если порт открыт — получает SYN-ACK. Если закрыт — RST-ACK.

`Код:`
```
sudo nmap -sS 192.168.10.140
```

2. FIN сканирование
Отправляет TCP-пакет с флагом FIN. Закрытые порты отвечают RST, открытые — игнорируют.
Это техника скрытного сканирования портов, которая отправляет пакеты с установленным только FIN-флагом, чтобы определить открытые порты, обходя многие брандмауэры, так как такие пакеты часто не блокируются;

`Код:`
```
sudo nmap -sF 192.168.10.140
```

3. Xmas сканирование
Отправляет TCP-пакет с флагами FIN, PSH, URG (как "рождественская ёлка"). Закрытые порты отвечают RST, открытые — игнорируют.
`Код:`
```
sudo nmap -sX 192.168.10.140
```

4. UDP сканирование
Отправляет UDP-пакет. Если порт открыт — служба может ответить (редко). Если закрыт — получает ICMP "port unreachable".
Медленнее TCP-сканирования, так как отсутствие ответа может означать открытый порт, и часто используется вместе с TCP-сканированием (например, -sS -sU) для проверки обоих протоколов. 
`Код:`
```
sudo nmap -sU 192.168.10.140
```
